# Cosmic Portfolio

This is a space-themed personal portfolio.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the development server:
   `npm run dev`

## Deploying to GitHub Pages

1. Run the build command to generate static files:
   `npm run build`
2. Deploy the contents of the `dist` directory to the `main` branch of your `jakubrollo.github.io` repository. Since this is a user GitHub page, the compiled files need to sit right at the root.
